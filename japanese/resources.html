Japanese Learning Resources

Here is the list of resources that I use or have used in my Japanese learning. Typically, a Google search will lead you to one or more these resources if you look up something like "Japanese [grammar rule]".


Tae Kim's Guide

This is the first resource I ever used and it is amazing (imo). I know that a lot of people like "structured" learning resources like Genki, but this way works so much better for me. I do not generally like a super structured language learning textbook that starts off with basic phrases and then explains different stuff with it. I like just learning the rules needed in order to speak the language and expand as I go. There is also an iOS and Android app that is very useful.


Jisho

Jisho is a very useful dictionary that I use quite a lot while reading things online. It provides extensive definitions that sometimes go beyond what is in the default Rikaichan (or other) dictionary files.


NHK Easy News

More recently, I have used NHK Easy News for reading practice. After only reading a few articles, I have already noticed my sentence comprehension ability growing. It also provides furigana and uses simple enough words. I do find myself having to look up a couple words from time to time (I'm not even past JLPT 5 vocab officially), but, in a given article, there is actually few diversity of words. I like this because it helps me focus on sentence structure and grammar rather than looking up every single word in a sentence.


Imabi

I have not actually used this site very extensively. However, after looking through the topics that are covered and the depth they are covered in, this is a very good resource in case you need an extra explanation.


Maggie Sensei

Lately, as I have been making more of an effort to read Japanese text, I have been stumbling upon Maggie Sensei's website should I need any explanation of a grammar rule. Normally, this site will show up in the first five results if you Google "Japanese [grammar rule/particle/verb conjugation]". The explanations and examples are pretty useful and can be invaluable should you need a quick explanation in the middle of reading something.


Rikaichan

A very useful dictionary plugin for FireFox. There is an extended version called Rikaisama and a Chrome port called Rikaikun, though I just use Rikaichan.


A Dictionary of Basic Japanese Grammar

As I mentioned earlier, I don't like a standard textbook explanation format. It is typically too long-winded for me. This book is very logical. It has no actual order of grammar rules (it is alphabetical) so you can easily just look up a nice, clean definition of a grammar function very easily. I often find myself just flipping through it if I am just bored and want to expand my general knowledge. Now, not that I am condoning any illegal action here, but you can -ahem- "acquire" a PDF version of this book should you not want to buy it pretty easily (I'm gonna be honest here: I have yet to buy a physical copy).


Anki

Anki is a very powerful flash card tool. I have been using it with various JLPT decks and the Core 2k deck and it works for me pretty well. However, I have recently been moving back to Memrise.


Memrise

Memrise is probably the most essential tool to the expansion of my Japanese reading level. Anki can help review, but I strongly recommend Memrise for base Japanese vocab learning. Using the various JLPT decks, it starts off with giving you a set of 25 words or so in hiragana or katakana, and then moves on the the same set of words, but in kanji. This helps to really drill down the meaning and recognition of words. It does this through interaction and different modes of memorizing. Some of these include: multiple choice, typing, and audio recognition. The JLPT N5 and N4 sets are each about 50-60 levels, so 2 levels a day + review would be about a month to learn one full level of words! The mobile app is great too.


JP Drills

I have only used this site a little bit, but it looks pretty promising. It provided Japanese quizes on different things such as grammar and vocabulary. Good for practice and seeing what you need to learn.
